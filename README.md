# AWS × Kiro Hackathon · Challenge Briefing

## Challenge 7 · Beginner Challenge
# DeskMate: AI FAQ Bot for New Employees
*A friendly onboarding chatbot that answers common workplace questions so new starters feel supported from day one.*

---

### PERSONA

An HR coordinator at a growing company who fields the same 50 questions from new starters every week — Wi-Fi passwords, leave policies, expense processes, who to contact for IT. They want to hand new employees a single place to get answers instantly, without waiting for a reply.

### INPUTS

- Mock employee handbook (PDF/text)
- Company policies summary
- FAQ document (50 Q&As)
- Org chart / contact list

### WHAT TO BUILD (USING KIRO)

Use Kiro to spec and build a simple conversational bot that ingests the FAQ and handbook, answers natural language questions from new starters, and responds with a direct answer plus a reference to the source document section. If it cannot answer, it should gracefully direct the user to the right person. Stretch goal: add a thumbs up/down feedback button to flag unhelpful answers.

### OUTPUTS & DELIVERABLES

- Working chatbot demo
- 3 sample Q&A exchanges
- 3-min pitch
- Kiro spec file
- One 'graceful fail' example
